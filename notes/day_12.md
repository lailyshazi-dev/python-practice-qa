## Что понял

- `-v` показывает подробный вывод с именами тестов.
- `-q` уменьшает количество текста в выводе.
- `-ra` добавляет расширенную сводку по особым результатам тестов.
- `-m smoke` запускает только тесты с маркером `smoke`.
- `deselected` означает, что pytest нашел тесты, но не выбрал их для запуска из-за фильтра.

## Команды дня

```powershell
python -m pytest
python -m pytest -v
python -m pytest -q
python -m pytest -ra
python -m pytest -v -m smoke
```

## Новые слова

- `verbose` - подробный режим.
- `quiet` - тихий режим.
- `summary` - сводка.
- `selected` - выбранные тесты.
- `deselected` - найденные, но не выбранные тесты.

## Правило дня

Флаги запуска помогают получать ровно столько информации, сколько нужно для текущей задачи.
